**[ НАЗВАНИЕ ПРОЕКТА ]**

"Игра для молодежи"

**[ ПРОБЛЕМНОЕ ПОЛЕ ]**

В настоящее время молодежь часто обходит стороной банковские темы, считая их скучными и сложными. Вследствие, современное поколение не обладает необходимыми знаниями финансовой грамотности для успешного и безопасного использования банковских продуктов. Приложение должно повысить вовлеченность в продукты Альфа-банка и научить пользователей базовым принципам финансовой грамотности. Игра должна показать, что финансы могут быть интересными. 

**[ ОБРАЗ ПРОДУКТА ]**

Разработка обучающей игры для молодежи внутри приложения Альфа-банка: разработать концепцию постоянной игры для молодёжи (14-22 года), которая повысит вовлеченность в продукты Альфа-Банка и обучит базовым принципам финансовой грамотности.

Пользователь видит игру в приложении альфа-банка. 
Внутри приложения альфа-банка пользователь замечает иконку, которая переводит его на новую страницу в игру. При переходе в игру пользователю предлагается выбрать персонажа(и иного), кратко объясняются правила и предлагается начать игру. 
1) Отказ. Пользователя перебрасывает на страницу, где система мотивации и поощрения работает наоборот, мотивируя его сыграть и демонстрируя, что отказ от игры - плохо
2) Он играет в игру. По завершении игра благодарит пользователя и хвалит его. Работает система мотивации и поощрения, после пользователь отправляется на страницу назад

Идея и механика игры для молодежи в Альфа-Банке (сюжет, правила, прогресс, награды); формат, в котором молодежь будет изучать финансовый контент; перечень ключевых тем; список игровых заданий и активностей, связанных с продуктами банка, которые помогут участникам закрепить знания и повысить вовлеченность; система мотивации и поощрений
