**[ НАЗВАНИЕ ПРОЕКТА ]**

"Игра для молодежи"

**[ ПРОБЛЕМНОЕ ПОЛЕ ]**

Приложение должно повысить вовлеченность в продукты Альфа-банка и научить пользователей базовым принципам финансовой грамотности. Молодежь часто обходит стороной банковские темы, считая их скучными и сложными.  Игра должна показать, что финансы могут быть интересными, повысить вовлеченность в продукты Альфа-банка и научить пользователей базовым принципам финансовой грамотности.

**[ ЗАКАЗЧИК / ПОТЕНЦИАЛЬНАЯ АУДИТОРИЯ ]**

Целевая аудитория: пользователи приложения Альфа-банка в возрасте от 14 до 22 лет 
Заказчик: Альфа-банк

**[ ЭТАПЫ РАЗРАБОТКИ ]**

1) Подготовка на основе анализа ЦА 2-3 черновых механик игры (концепт + core-механика)
2) встреча – просмотр анализа и черновых концептов с ментором 
3) доработка концептов по комментариям 
4) Выбор командой приоритетного концепта
5) встреча с ментором, выбор одного концепта, его обсуждение, выстраивание плана работы
6) Проработка концепта – развитие сюжета, стилистики, UX-логики, ретеншн-механик, обучающих форматов, списка заданий
7) Создание кликабельного прототипа игры с mvp дизайна
8) Первичное тестирование прототипа
9) Корректировка прототипа по полученной обратной связи
10) Рассмотреть идеи проведения минимальных изменений и переиспользования для других сегментов клиентов
